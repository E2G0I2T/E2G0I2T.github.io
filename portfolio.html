안드로이드 일단어 앱

안드로이드 스튜디오에서 JAVA와 SQLite를 활용하여 만든 일단어 공부 앱입니다

자세히 보기

안드로이드 스튜디오에서 JAVA와 SQLite를 활용하여 만든 일단어 학습 앱입니다.
JLPT 시험을 위한 단어 학습 목적으로 만들었으며, 총 7000여개의 단어, 발음, 의미를 db에 저장한 뒤, 급수별로 테이블을 만들었습니다.
앱에서는 급수별로 단어를 외웠는지 아닌지를 체크하여 아는 단어나 모르는 단어를 선택해 학습할 수 있고, 단어의 발음이나 의미를 학습하는 기능도 있습니다.
현재 PlayStore에 'JLPT 단어 한 입' 이라는 앱으로 출시되어 있습니다.

안드로이드 gallag 게임

팀 프로젝트로 만든 게임으로 간단한 슈팅 게임 입니다

자세히 보기

게임 실행 시 비행기 선택 가능하며, 비행기를 선택하면 화면 중앙에 버튼이 생성되어 버튼 클릭 시 게임이 시작됩니다.
조이스틱으로 이동 가능(화면 가장자리에 부딪힐 시 더 이상 화면 바깥방향으로 이동 불가)합니다.
총알 발사, 필살기 버튼을 클릭하여 적 공격 가능하며, 적을 처치 시 포인트가 상승합니다.
재장전 버튼으로 총알을 장전할 수 있습니다.
아이템을 먹으면 특정한 효과가 발생합니다.
파란색 : 총알 업그레이드 (총 8단계)
빨간색 : 라이프 1 획득
노란색 : 속도 업그레이드
일시 정지 버튼으로 일시 정지 가능하며 일시 정지 화면에서 배경 음악, 효과음 ON/OFF 가능합니다.
적의 총알에 맞거나 적과 부딪힐 시 라이프 1 감소됩니다.
라이프가 모두 감소되면 게임 종료 (라이프 총 3)됩니다.
